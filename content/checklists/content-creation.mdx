items:
  - Planning Stage
    - Identify target audience
    - Define content goals
    - Research topics and keywords
    - Create content outline
  - Content Creation
    - Write draft content
    - Create visuals and media
    - Optimize for SEO
    - Edit and proofread
  - Review Stage
    - Get peer review
    - Check for accuracy
    - Ensure quality standards
    - Test if applicable
  - Publishing
    - Prepare for publication
    - Schedule publication time
    - Publish content
    - Promote across channels
  - Post-Publish
    - Monitor engagement
    - Respond to comments
    - Track performance metrics
    - Plan for content refresh
